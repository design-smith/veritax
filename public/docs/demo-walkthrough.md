# Demo Walkthrough

## Order

1. Briefing
2. Graph
3. Findings
4. Library / Document viewer
5. Gathering

## Shape

Briefing and Graph establish that the Record is alive. Findings and the Library viewer carry the contradiction-and-trust moment. Gathering is the single future-facing screen: messy source material becomes the Record.

## Removed From Friday Cut

- Factory
- Monitor
- Calendar
- Standalone Ask page
- Runs
- Admin
- Connectors
- Settings
- External portals
- Benchmark Studio

Ask remains available as an overlay behavior from the shell, not as a standalone stop.

## Known Mocked Boundaries

- Data is fixture-backed.
- Commands are demo envelopes, not backend writes.
- Gathering uses local demo state and telemetry stubs.
- Citation clicks open seeded documents and highlighted spans.
